Navbar

This navbar is responsive. check it out ;)

            <nav class="navbar bg-light navbar-light">
        <a href="#" class="logo">Logo</a>
        <ul class="navbar-links">
            <li><a href="#">Home</a></li>
            <li><a href="#">About Us</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>

To change navbar's text colors, you need to use classes: text-success, text-danger, text-warning and text-info on the <nav> tag.

To change navbar's background color, you need to use classes: bg-light, bg-dark, bg-success, bg-danger, bg-warning and bg-info on the <nav> tag.

Or you can style it by yourself using css :) don't forget to add !important after every line in css
